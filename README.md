# messy

Discover Unity with a small rogue-like spaceshooter.

## Development keys
Press keys from 1 to 7:

1 - make a boss appear
2 - add a pistol
3 - add a rifle
4 - add a shotgun
5 - add a rocket launcher
6 - add a laser rocket launcher
7 - add a koopa hammer thrower
